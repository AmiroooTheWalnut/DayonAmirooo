Download

Auf der folgenden Seite finden Sie die aktuellsten Installationspakete für ihre Plattform.

Historische Versionen (v1.0 to v1.2) finden Sie da.

Windows: es existieren zwei Installationspakete, die eigentliche Installation ist selbsterklärend. Sie haben die Wahl zwischen einem Installer mit oder ohne integrierter JRE (OpenJDK 1.8 basiert). Das Paket mit integrierter JRE eignet sich bloss für 64Bit Windows Varianten.

Linux: das .tgz-Paket mittels tar xzvf entpacken und anschliessend die Shellskripte (bin/dayon*) mit chmod +x ausführbar machen. Die installierte JRE resp. JDK wird automatisch detektiert. Die JRE ist nicht im .tgz-Paket enthalten. Sollte die Autodetektierung aus irgend einem Grund fehlschlagen, so überprüfen Sie das Vorhandensein einer JRE und definieren deren Pfad in der Variable JAVA_HOME im Skript (bin/dayon_run.sh). Beachten Sie, dass die Skripte bloss auf diversen Debian und Ubuntu Versionen getestet worden sind.

Mac OS/OS X: Die Applikation ist bisher nicht auf einem Mac getestet worden. Eigentlich sollten die (Linux-) Shellskripte dort ebenso funktionieren (Voraussetzung: installierte JRE).

Änderungen Dayon! v1.3 Phoenix (10. Dezember 2016)

  • Probleme mit Windows UAC-Dialogen behoben.
  • Unterstützt hohe Bildschirmauflösungen, welche in früheren zu schwerwiegenden Fehlern geführt hatten.
  • Weniger manuelle Konfiguration unter Linux notwendig - die installierte JAVA-Version wird automatisch erkannt.
  • Neuer, verbesserter Installer für Windows, inklusive einer OpenJDK (1.8) basierten JRE.
  • Aktualisierte Bibliotheken.
  • GPLv3.

  • Öffentliche IP-Adresse-Dienst (dayonhome.sourceforge.net/whatismyip.php).
  • Detailliertere Schnellstart-Anleitung.
  • Französische Lokalisierung.
