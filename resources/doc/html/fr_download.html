Télécharger

Utilisez cette page pour télécharger la version qui correspond à votre OS (i.e., Windows, Linux, Mac, ...).

Windows: un programme d'installation est disponible et l'installation devrait être assez directe (vous pouvez si vous le preferez utiliser le fichier ZIP). Notez que deux versions sont disponibles: une avec le JRE 1.6 de Sun inclu, et l'autre sans (l'application essaiera de localiser sur la machine un JRE 1.6).

Linux: you devez utiliser le fichier ZIP et modifier peut-être chmod +x des scripts (bin/dayon*) ainsi que la location du JRE/JDK 1.6 (bin/dayon_run.sh). Notez, que j'ai pu simplement testé les scripts mais pas l'application car je n'ai pas d'accès a une machine autrement qu'en mode terminal. Dites moi si quelque chose ne fonctionne pas.

Mac OS: Je n'ai pas pu tester l'application sur un Mac (aucun accès) mais j'imagine que vous pouvez utiliser les shell scripts comme sur Linux.

Change Log

Changes for Dayon! v1.2 (January 3rd, 2009)

  • Fix possible black screen (assistant) at startup.
  • Control support.
  • Java Web Start (v1.2 is not compatible with v1.1) to start the assisted.
  • Public IP address service (dayonhome.sourceforge.net/whatismyip.php).
  • Quickstart guide improvements.
  • French localization.

Changes for Dayon! v1.1 (December 24, 2008)

  • Protocol improvements (v1.1 is not compatible with v1.0).
  • Improved a bit the CPU usage.
  • Cleanup initial screen capture encoding.
  • New compression methods : BZIP2, LZMA.
  • Bitmap cache usage.
  • The GUI is a bit more polished.

Changes for Dayon! v1.0 (December 4, 2008)

  • Initial release.
