Download

Use the following page to download the lastest package according to your platform.

Legacy releases (v1.0 to v1.2) can be downloaded here.

Windows: an installer is available and the installation should be straightforward. The package comes bundled with an OpenJDK 1.8 based JRE.

Linux: you have to tar xzvf the package and then setup the chmod +x of the scripts (bin/dayon*). The location of the installed JRE/JDK is going to be detected automatically. It is not bundled in the tgz. If it should for whatever reason fail to do so, then you can set the JAVA_HOME in (bin/dayon_run.sh) manually. Note that, I've been testing this on Debian and Ubuntu distros only.

Mac OS: I've not tested the application on a Mac platform (no access to such a machine) but I guess you can use the shell scripts.
