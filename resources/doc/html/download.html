Download

Use the following page to download the package according to your platform.

Windows: an installer is available and the installation should be straightforward; you might as well use the zip package if you prefer. Two packages are availabe: one with the Sun JRE 1.6 bundled, the other one without (the exe is going to locate a JRE 1.6 on the machine before starting).

Linux: you have to unzip the package and then setup the chmod +x of the scripts (bin/dayon*) as well as the location of a JRE/JDK 1.6 (bin/dayon_run.sh). Note that, I've simply tested the scripts but not the application itself as I've only access to a machine in terminal mode. Please tell me if something does not work.

Mac OS: I've not tested the application on a Mac platform (no access to such a machine) but I guess you can use the shell scripts.
