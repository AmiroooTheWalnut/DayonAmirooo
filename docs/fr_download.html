Télécharger

Utilisez cette page pour télécharger la version qui correspond à votre OS (i.e., Windows, Linux, Mac, ...).

Les anciennes versions (v1.0 à v1.2) peuvent être téléchargées depuis ici.

Windows: deux programmes d'installation sont disponible et l'installation devrait être assez directe. Un paquet est livré avec un JRE à base du OpenJDK 1.8, l'autre sans. Cet avec JRE peut seulement etre utilisé avec des versions de Windows de 64bit.

Linux: extraisez le fichier TGZ. Rendez le script setup.sh exécutable (chmod +x) et exécutez-le. La location du JRE/JDK installé va être détecté automatiquement. Il n'est pas livré dans le tgz. Si, pour quelque raison que ce soit, il ne le fasse pas, vous pouvez configurez JAVA_HOME dans (bin/dayon_run.sh) manuellement. Notez que, j'ai testé ceci sur Debian et Ubuntu seulement.

Mac OS/OS X: extrayez le fichier TGZ à une location de votre choix. Pour plus de commodité, vous pouvez créer un alias pour bin/dayon_assisted et bin/dayon_assistant et les copier sur votre bureau. (JDK > 1.6 nécessaire)

Journal des modifications

Modifications pour Dayon! v1.5 Pendragon (5 mai 2017)

  • Support IPv6
  • OS X compatible scripts
  • JRE actualisé

Modifications pour Dayon! v1.4 Skytale (3 janvier 2017)

  • Communication crypté entre l'assistant et l'assisté (v1.4 n'est pas compatible avec toutes les anciennes versions)
  • Installeur pour Windows optimisé
  • Localisation allemande

Modifications pour Dayon! v1.3 Phoenix (10 décembre 2016)

  • Résolu les problèmes avec Windows UAC-dialogues
  • Supporte les résolutions d'écran énormes, qui causaient des erreurs fatales sur les versions précédentes
  • Moins de configuration manuelle sous Linux nécessaire - la version installée de JAVA est maintenant détecte automatiquement
  • Nouvel installeur amélioré pour Windows, y compris un JRE à basé du OpenJDK (1.8)
  • Bibliothèques mises à jour
  • GPLv3

Modifications pour Dayon! v1.2 (3 janvier 2009)

  • Correction d'un écran noir possible (assistant) au démarrage
  • Support de contrôle
  • Java Web Start (n'est pas compatible avec v1.1 ) pour lancer l' assisté
  • Service d'adresse IP publique (dayonhome.sourceforge.net/whatismyip.php)
  • Améliorations au guide de démarrage rapide
  • Localisation française

Modifications pour Dayon! v1.1 (24 decembre 2008)

  • Améliorations apportées au protocole (v1.1 n'est pas compatible avec v1.0)
  • Amélioré un peu l'utilisation du processeur
  • Nettoyage de l'encodage initial de capture d'écran
  • Nouvelles méthodes de compression: BZIP2, LZMA
  • Utilisation du cache de bitmap
  • L'interface graphique est un peu plus polie
