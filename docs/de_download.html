Download

Auf GitHub und SourceForge finden sie die aktuellsten Installationspakete für ihre Plattform.

Seit Version 1.10.2 ist Dayon! auch im Snap Store verfügbar:

Windows: ab Version 1.9 existieren vier unterschiedliche Installationspakete, die eigentliche Installation ist selbsterklärend. Sie haben die Wahl zwischen einem Installer mit oder ohne integrierter JRE (OpenJDK 1.8 basiert). Das Paket mit integrierter JRE eignet sich bloss für 64Bit Windows Varianten. Daneben existieren zwei separate "Quick Launch" Versionen für den Assistenten und de Assistierten. Diese beiden portablen Versionen benötigen keine Installation. Um bei diesen Versionen möglichen Probleme mit der Benutzerkontensteuerung zu vermeiden, wird empfohlen den Assistierten mit Administratorenrechten zu starten.

Debian & Ubuntu: die .snap und .deb Pakete lassen sich am komfortabelsten mittels dem integrierten Software Installer installieren. Alternativ können sie es mittels dem Befehl snap install dayon, respektive sudo dpkg -i dayon*.deb, via Terminal installieren.

Linux: entweder die sich selbständig aktuell haltende .snap App via Snap Store oder mittels snap install dayoninstallieren - oder das .tgz Paket mittels tar xzvf entpacken. Im Zielverzeichnis das Shellskript setup.sh ausführen.
Die installierte JRE resp. JDK wird automatisch detektiert. Die JRE ist nicht im .tgz-Paket enthalten. Sollte die Autodetektierung aus irgend einem Grund fehlschlagen, so überprüfen sie das Vorhandensein einer JRE und definieren sie deren Pfad in der Variable JAVA_HOME im Skript (bin/dayon.sh). Beachten sie, dass die .tgz Pakete bloss auf diversen Debian und Ubuntu Versionen getestet worden sind.

Mac OS/OS X: das .tgz Paket an einen Ort ihrer Wahl entpacken. Ev. einen Alias für bin/dayon_assisted und bin/dayon_assistant erstellen und auf den Desktop kopieren. (Voraussetzung: installiertes JDK 1.8 oder neuer).
